# GYMPOINT

Gympoint is an application developed for gym management.

# What I used to develop this application:

- NodeJS using Express
- Javascript ES6
- Postgre SQL
- JWT (JSON Web Token)

# Tools
- Visual Studio Code (IDE)
- Nodemon and Sucrase
- ESLint (Patterns)
- Sequelize (ORM)
- Insomnia
